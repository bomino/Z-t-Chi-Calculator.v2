Instructor Mode

Build a signed problem link your students can open. They see the prompt, use the calculator, and get immediate feedback on whether their answer matches yours.

How it works: You describe the problem below, set the expected answer, and click Generate link. If this deployment has a backend configured, the link is cryptographically signed so students can't silently edit the expected answer. If no backend is configured, the link still works but carries an unsigned badge — fine for low-stakes practice, not recommended for graded assignments.

Instructor sign-in

This page can issue signed problem links only to authorized instructors. Paste the token your administrator gave you; it's stored in this browser's localStorage so you won't be asked again on this device. The token never leaves your machine except to verify against the backend.

Without a valid token you can still build problem links, but they will carry an unsigned badge. Students should be told not to trust unsigned links for graded work.
